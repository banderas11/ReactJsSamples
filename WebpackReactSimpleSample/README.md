# Webpack and React Together

## Running 

To run the thing, you need to install a few things:

 - Webpack: `npm install webpack -g`
 - http-server: `npm install http-server -g`

Now, you can get the sample running:

```bash
npm install
webpack
http-server dist/ -o
```
